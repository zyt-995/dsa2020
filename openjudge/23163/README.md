## P1040:判断无向图是否连通有无回路

- 

###### 总时间限制:

1000ms

###### 内存限制:

65536kB

###### 描述

例题：给定一个无向图，判断是否连通，是否有回路。

###### 输入

第一行两个整数n,m，分别表示顶点数和边数。顶点编号从0到n-1。 (1<=n<=110, 1<=m <= 10000)  
接下来m行，每行两个整数u和v，表示顶点u和v之间有边。

###### 输出

如果图是连通的，则在第一行输出“connected:yes",否则第一行输出“connected:no"。  
如果图中有回路，则在第二行输出“loop:yes ",否则第二行输出“loop:no"。

###### 样例输入

3 2
0 1
0 2

###### 样例输出

connected:yes
loop:no
