# 思路
Prim 算法：
visited 列表用于记录每个顶点是否已加入最小生成树。
key 列表用于记录每个顶点到最小生成树的最小权值。
从顶点 0 开始，每次选择距离最小生成树最近的顶点，并将其加入最小生成树。
更新与该顶点相邻的顶点的最小权值。
