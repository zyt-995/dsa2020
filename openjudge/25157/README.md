# P1140: 判断有向图是否有回路

## 总时间限制: 1000ms  内存限制: 65536KB

## 描述

判断有向图是否有回路

## 输入

第一行两个整数n(0 < n <= 110),m(0 <= m < 10000)，分别表示顶点数和边数。顶点编号从0到n-1。接下来m行，每行两个整数u和v，表示从顶点u到v有边。

## 输出

如果图中有回路，则"yes",否则输出"no"

## 样例输入

```
#样例1
3 2
0 1
0 2
#样例2
3 3
0 1
1 2
2 0
```


## 样例输出
```
#样例1
no
#样例2
yes
```
